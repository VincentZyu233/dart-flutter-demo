# Flutter UI Showcase - PoC 规划

## 项目定位

一个用来展示 Flutter UI 能力的简易 App，给同事演示用。不追求业务逻辑，纯粹展示 Flutter 在 UI 层面的控制力、跨平台一致性和高性能渲染。

App 名称: **Flutter Showcase**
目标平台: Windows x64, Linux x64, Android APK

---

## 页面规划 (7 个页面)

### 页面 1: 经典对话框实验室

展示 Flutter 对像素级还原的控制力。模拟一个 90 年代 Windows 桌面窗口环境。

背景色用经典的 Windows 灰 #C0C0C0。页面中央放一个复古对话框，标题栏是蓝底白字，写着 "A Short title is Best"。对话框左侧放一个带阴影的蓝色问号图标，右侧垂直排列两个按钮（OK 和 Cancel），按钮要有明显的 3D 边框阴影效果，模拟老系统的厚重感。中间放两个带内阴影的输入框。对话框下方再放一个触发按钮，点击弹出这个对话框。

重点展示: Flutter 不依赖原生系统组件，可以 100% 还原任何时代的 UI 规范。

---

### 页面 2: 文字排版工作室

展示 Flutter 强大的文本渲染引擎和字体自定义能力。

页面上半部分是一个大的预览区，中间显示一句全字母句 "The quick brown fox jumps over the lazy dog"。下方放三个 Slider 滑块，分别实时调节字号 FontSize、字间距 LetterSpacing、行高 LineHeight。滑块拖动时文字即时变化，不能有卡顿。再放一个 Toggle 开关，用于在系统默认字体和自定义衬线字体之间瞬时切换。底部可以加一个颜色选择器，改变文字颜色。

重点展示: 文本排版的平滑度，动态加载外部字体的便捷性。

---

### 页面 3: 响应式布局挑战

展示 Flutter 一次编写到处运行的响应式适配能力。

页面用 LayoutBuilder 监听屏幕宽度。内容是一个瀑布流风格的卡片列表，每个卡片包含一个随机彩色占位块（模拟图片）和一段 Lorem Ipsum 占位文案。手机竖屏时显示 1 列大卡片，旋转屏幕或在平板 PC 上查看时动态变为 2 列或 3 列。卡片之间有间距，整体看起来像一个内容聚合 Feed。

重点展示: App 在不同设备上自动调整布局，无需为每个平台重写样式。

---

### 页面 4: 动效与交互实验室

展示 Flutter 60/120 FPS 的渲染性能和状态管理逻辑。

页面是一个模拟注册表单。所有元素进入时带 FadeIn 和 Slide 动画，依次错开出现。输入框下方有实时校验文案，比如输入 "admin" 时文案立刻变红并跳动提示"用户名已存在"。右下角放一个浮动动作按钮 FAB，点击后弹出扇形菜单，展示 Flutter 的 Stack 层叠布局能力。表单提交时有一个从左到右的进度条动画。

重点展示: 高频动画的流畅度，通过状态管理实现复杂交互逻辑。

---

### 页面 5: 导航与层级实验室

展示 Flutter 的页面导航和层级切换能力。

顶部放一个 TabBar 标签页，分三个标签: Components、Layout、Settings。左侧放一个 Drawer 侧边抽屉，抽屉里有用户头像、几个菜单项和一个版本号。底部放一个 BottomNavigationBar 底部导航栏，三个图标分别对应三个不同的页面内容。点击 Tab 或底部导航时内容平滑切换，可以用 Hero 动画让元素在页面间飞过去。

重点展示: Flutter 在页面切换时的平滑过渡动画和多种导航模式的共存。

---

### 页面 6: 数据流与列表展示

展示 Flutter 处理大规模重复数据的能力。

页面主体是一个长列表，带下拉刷新 RefreshIndicator。列表里每个 item 是一个卡片，包含随机生成的用户名、一段 Lorem Ipsum 摘要和一个彩色头像占位。列表支持无限滚动，滚动到底部自动加载更多数据，加载时底部显示一个 circular loading indicator。页面右上角有一个按钮，点击切换为 GridView 网格布局。

重点展示: 即使有几千个 item，Flutter 的渲染也不会卡顿，列表滚动丝滑。

---

### 页面 7: 控件与反馈实验室

展示 Flutter 丰富的内置控件和实时反馈机制。

页面分几个区域。第一区域是选择器: 一组 Radio 单选框、一组 Checkbox 复选框、几个 Switch 开关。第二区域是进度反馈: 一个 LinearProgressIndicator 线性进度条和一个 CircularProgressIndicator 圆形加载环，旁边放一个按钮可以启动/暂停进度动画。第三区域是提示反馈: 点击按钮弹出 SnackBar 轻提示，或者弹出 BottomSheet 底部面板。所有控件的状态变化都要有视觉反馈，比如颜色变化、数值变化。

重点展示: 实时反馈的交互设计，控件状态的即时响应。

---

## 技术选型

- 框架: Flutter (最新稳定版)
- 状态管理: 先用 setState 起步，页面多了再考虑 Provider 或 Riverpod
- 路径: 底部导航 + TabBar 组合导航
- 字体: 系统默认字体为主，页面 2 用 Google Fonts 插件加载自定义字体
- 占位数据: 用 dart:math 随机生成，不需要 mock 库

## 构建目标

1. `flutter build windows --release` → Windows x64 可执行文件
2. `flutter build linux --release` → Linux x64 可执行文件
3. `flutter build apk --release` → Android APK

## 开发顺序建议

页面 1 (经典对话框) → 页面 7 (控件反馈) → 页面 2 (文字排版) → 页面 4 (动效交互) → 页面 3 (响应式布局) → 页面 6 (数据列表) → 页面 5 (导航层级)

从简单静态页面开始，逐步增加交互复杂度。
